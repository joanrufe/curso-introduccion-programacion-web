Objetos y tipos avanzados de datos

Los objetos son estructuras de datos que se componen de propiedades. Cada una de estas propiedades tiene un nombre y pueden contener cualquier tipo de dato u otros objetos. Los objetos también pueden contener functiones, en cuyo caso las llamados métodos.

El modelo de objetos en javascript se basa en la herencia prototipica . Permite a un objeto puede heredar de otros y usar los métodos definidos en algúno de sus anscestros. No suele ser habitual crear prototipos a la hora de desarrollar páginas web pero si es importante entender cómo funciona la herencia prototípica para entender cómo funciona de javascript.

Muchas de las herramientas necesarias para el desarrollo web que tenemos a nuestra disposición en javascript son objetos. Como el objeto document el cual es la interfaz para la manipulación del DOM (representación estructural del documento HTML) o el objeto window que representa la ventana del navegador.
