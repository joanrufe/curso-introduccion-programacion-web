Modificando el DOM

Antes de manipular el DOM necesitamos seleccionar los elementos que queremos modificar. Para ello disponemos de varias funciones como getElementById o getElementsByClassName. Otras funciones más versátiles son querySelector y querySelectorAll que seleccionan elementos usando selectores CSS. Todas estas funciones devuelven el objeto correspondiente del DOM, el cual podemos manipular para cambiar su contenido o añadir eventos como veremos en el siguiente capítulo.
