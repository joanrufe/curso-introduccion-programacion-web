Diseño responsive

A la hora de diseñar una web tenemos que tener en cuenta que nuestra aplicación podrá ser accedida desde distintos dispositivos. Es importante añadir estilos para que se vea bien en un ordenador de sobremesa, un portátil, un tablet o un móvil. Esto se llama diseño responsivo .

Para adaptar una web para diferentes dispositivos usamos las media queries . Estas expresiones en CSS nos permiten aplicar estilos dependiendo del ancho de la pantalla del dispositivo (o la ventana del navegador). Se componen de la palabra clave @media seguido de una serie de condiciones 

    @media (max-width: 600px) {
        .barra_lateral {
            display: none;
        }
    }
