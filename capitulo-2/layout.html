Diseñando una web

A la hora estructurar una web en bloques y secciones usaremos determinadas etiquetas como div, section, header o footer entre otros. Esto sirve como base pero para diseñar webs necesitamos posicionarlos adecuadamente en la pantalla.

En CSS disponemos de varios mecanismos para ordenar visualmente el contenido al contenido.

Los elementos en linea se muestran en la misma linea de un texto. Por defecto las etiquetas como a, span, em o b se muestran en linea.

Los elementos flotantes son etiquetas a las que se le aplica la propiedad CSS float.

Flexbox es otro mecanismo para ordenar los elementos visualmente.
