Empezando con CSS

La sintaxis de CSS se compone de un selector y un bloque de estilos. A todos los elementos que coincidan con el selector se les aplicará el bloque de estilos. 

    pre {
      background: gray;
      padding: 10px 0;
      color: white;
      max-width: 500px;
      font-weight: bold;
    }

    #caja1 {
      background-color: red;
    }

    .caja {
      width: 50px;
      height: 50px;
      border: 1px solid blue;
    }
      <div id="caja1" class="caja"></div>
      <div class="caja"></div>

En HTML existen muchos atributos diferentes pero existen dos especialmente usados en CSS: id y class. Estos attributos se representarían en el selector como #caja1 y .caja. Las siguientes cajas son un ejemplo de selectores.
