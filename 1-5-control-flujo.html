Control de flujo

Javascript soporta un conjunto de sentencias para el manejo del flujo de ejecución de una aplicacion web.

Las sentencias condicionales if y else nos permiten hacer que la ejecución de un programa ejecute un conjunto de sentencias u otro.

Los bucles se usan para repetir un conjunto de sentencias mientras se cumpla una condición.
