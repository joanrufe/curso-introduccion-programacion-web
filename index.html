Introduccion a la programacion Web

Este curso pretende servir como herramienta didactica para la iniciación a la programación web y no como guía detallada de un tema tan extenso. Recomiendo leer los enlaces proporcionados para tener recursos de aprendizaje de más calidad. La comunidad de desarrolladores de Mozilla tiene recursos de aprendizaje muy valiosos.

Las páginas o aplicaciones web son un tipo de aplicaciones que se ejecutan en el navegador web (Chrome, Firefox, etc...). Cuando un usuario introduce una dirección web (ej: https://www.example.com ) en el navegador, este realiza una petición al servidor web usando el protocolo HTTP . El servidor responde enviando un documento HTML que será procesado por el navegador del usuario y mostrado en la pantalla del dispositivo (pc, tablet, móvil, etc...).

Reformular (cambiar orden) esta frase: HTML , JavaScript y CSS son los 3 lenguajes imprescindibles para el desarrollo de aplicaciones web y que aprenderas a lo largo del curso. HTML hace referencia al lenguaje de marcado que define el contenido de una web (por ejemplo las cabeceras, parrafos, imágenes o enlaces) usando una serie de etiquetas anidadas (ejemplos: <h1></h1>, <p></p>, <img></img>, <a></a>). Añadiendo hojas de estilo (CSS) al documento se personaliza la manera en que se presenta el contenido. Y ejecutando arhivos Javascript proporciona dinamismo e interactividad a las aplicaciones Web modernas.
